#### 反转矩阵
```java
/*
 * [[1, 2, 3],       [[1, 4, 7],
 *  [4, 5, 6],   =>   [2, 5, 8],
 *  [7, 8, 9]]        [3, 6, 9]]
 */
public int[][] transpose(int[][] A) {
    int[][] newArray = new int[A[0].length][A.length];
    for (int i = 0; i < A.length; i++) {
        for (int j = 0; j < A[i].length; j++) {
            newArray[j][i] = A[i][j];
        }
    }
    return newArray;
}
```
